КУРСОВАЯ РАБОТА
по математической логике и теории алгоритмов
на тему:
«Оценка различия текстовых строк»

Выполнила: студентка 2 курса 231 группы Забелина Татьяна Сергеевна.

В ходе разработки были изучены алгоритмы для вычисления степени различия двух текстовых строк. Результатом теоретических исследований стала программная реализация алгоритма Вагнера-Фишера на языке программирования Java.
Курсовая работа была написана с помощью системы компьютерной верстки LaTex. Для запуска программы необходимо в командной строке ввести команду: java -jar dictionary.jar.

Файлы и папки:
Main.java - исходный код программы - главный класс с функцией main()
Dictionary.java - исходный код программы - класс Словарь, объект которого создается в функции main()

dictionary.jar - исполняемый файл

nameList.txt - файл со списком имен
Evaluation_of_differences_of_text_strings.pdf - текст работы
